WideWorldImporters SQL Analysis

SQL Server data analysis project using the WideWorldImporters sample database

SQL Business Analysis SQL Server Window Functions Churn Analysis

Project Overview

This project focuses on writing analytical SQL queries using the Microsoft WideWorldImporters sample database. The goal was to practice working with a larger relational database and answer business-oriented questions using SQL.

The project was completed as part of a Data Analyst course assignment. The original assignment included multiple SQL analysis tasks. In the course context, it was acceptable to submit completed tasks rather than publishing every assignment prompt.

Database

WideWorldImporters is a Microsoft SQL Server sample database that contains business entities such as customers, orders, invoices, stock items, suppliers, cities, countries and customer categories.

Analysis Areas

  • Annual sales and income growth analysis.
  • Top customers by quarter.
  • Most profitable products.
  • Nominal product profit ranking.
  • Supplier product aggregation.
  • Top customers by total spending and geographic location.
  • Monthly order matrix.
  • Potential customer churn detection.
  • Customer category business risk analysis.

Tools & Technologies

  • Microsoft SQL Server
  • SQL Server Management Studio
  • T-SQL
  • WideWorldImporters sample database

SQL Concepts Used

  • Joins across multiple tables.
  • Aggregations and grouped calculations.
  • Common Table Expressions.
  • Window functions.
  • Ranking functions.
  • Date functions.
  • Conditional logic.
  • String aggregation.
  • Pivot-style reporting.

Business Value

The queries in this project demonstrate how SQL can be used to turn transactional data into business insights. The analysis supports questions related to sales growth, customer value, product profitability, supplier relationships, churn risk and customer concentration risk.

What I Learned

Through this project, I practiced writing analytical SQL queries, joining multiple business tables, calculating business metrics, ranking customers and products, detecting potential churn patterns and structuring SQL analysis for a public portfolio.
