NBA Player Performance Analysis

Python exploratory data analysis project focused on NBA player statistics

Python pandas Matplotlib Seaborn EDA Correlation Analysis

Project Overview

This project analyzes NBA player performance data using Python. The goal was to explore scoring patterns and understand which performance factors are most related to the number of points scored by a player.

The analysis focuses on the relationship between points scored, minutes played, field goal percentage, assists, rebounds and efficiency. The project includes data loading, initial exploration, missing value handling, descriptive statistics, visualizations, correlation analysis and grouped performance analysis.

Dataset

The dataset contains individual NBA player statistics. Each row represents one player and includes performance metrics such as points, minutes, assists, rebounds, field goal percentage and efficiency rating.

Main Columns

  • PLAYER — player name.
  • PTS — total points scored.
  • MIN — minutes played.
  • FG_PCT — field goal percentage.
  • AST — assists.
  • REB — rebounds.
  • EFF — efficiency rating.

Analysis Process

  • Loaded the dataset into Python using pandas.
  • Explored the structure and content of the dataset.
  • Handled missing values in a controlled way.
  • Calculated descriptive statistics.
  • Created multiple visualizations to identify patterns.
  • Analyzed correlations between performance metrics.
  • Grouped players by minutes played to confirm scoring trends.

Visualizations

  • Histogram showing the distribution of points.
  • Scatter plot showing the relationship between minutes and points.
  • Bar chart showing the top 10 players by points.
  • Boxplot for detecting outliers in scoring.
  • Correlation heatmap for numeric performance metrics.
  • Line chart showing the trend between minutes and points.
  • Grouped bar chart showing average points by minutes range.

Key Findings

  • Most players score a relatively low number of points.
  • A smaller group of players achieves much higher scoring values.
  • Players who play more minutes generally score more points.
  • Points are strongly related to both minutes played and efficiency.
  • Grouped analysis confirms that higher playing time is associated with higher average scoring.

Tools & Technologies

  • Python
  • pandas
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

What I Learned

Through this project, I practiced working with a real dataset, cleaning and exploring data, creating clear visualizations, analyzing statistical relationships and turning raw player statistics into structured insights.
